# OpenEpl.ELibInfo
一个用于读取易语言支持库信息的 NuGet 库（C#）  
[![NuGet](https://img.shields.io/nuget/v/OpenEpl.ELibInfo.svg)](https://www.nuget.org/packages/OpenEpl.ELibInfo)  
命名空间：`OpenEpl.ELibInfo`  

## 运行条件
.NET Core 2.0 或更高（不支持在 XP 环境下运行）  
推荐在 Windows 系统上运行
